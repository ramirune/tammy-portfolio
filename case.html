Case Study for myFlix-Angular Application

Objective

Building a client-side for an application myFlix by using Angular based on the existing serverside code (REST API and database) from myFlix app's server-side, Achievement2.

The application provides users with access to information about different movies, directors, and genres. User will be able to sign-up, update their personal information, and create a list of their favorite movies.

Purpose&Context

myFlix was an Acheivement project in Full-Stack Immersion course at Careerfoundry. In this project I was be able to use Angular to build the client-side interface for a new single-page, responsive app, that includes several interface views that will handle data through the previously-defined REST API endpoints. I used Agile project management techniques including user stories, story points and a Kanban board which are commonly used by companies in the tech-industry.
